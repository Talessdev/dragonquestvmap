# Dragon Quest V — Mapa Interativo de Monstros

Mapa interativo dos 202 monstros de **Dragon Quest V: Hand of the Heavenly Bride** (versão PlayStation 2 com tradução de fãs em inglês). Mostra onde cada monstro aparece no mapa-múndi, seus drops (com taxas reais) e a chance de recrutamento.

Disponível em **português, inglês e espanhol**.

## Funcionalidades

- Mapa-múndi interativo com 40 localizações
- Catálogo completo de 202 monstros com sprites
- Itens dropados e respectivas taxas (1/8, 1/16, 1/256...)
- Taxas reais de recrutamento (apenas para monstros recrutáveis)
- Suporte a 3 idiomas (PT / EN / ES)
- Layout responsivo

## Tecnologias

- React 19 + TypeScript
- Vite 7
- Tailwind CSS 4
- Wouter (rotas)
- Radix UI

Pré-requisitos: **Node.js 20 ou superior**.
